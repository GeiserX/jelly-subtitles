Quick Start

WhisperSubs can download whisper-cli binaries directly from the plugin settings page. These pre-built binaries require runtime libraries that may not be included in your Jellyfin container.

Running on bare metal? If Jellyfin runs directly on your system (not in Docker), the required libraries are usually already present. Just select your variant and download — the plugin will tell you if anything is missing.

Variant Requirements

Every variant requires libgomp1 (the OpenMP threading library). GPU variants need additional libraries.

Variant Required Packages Install Command
CPU libgomp1 apt install libgomp1
Vulkan libgomp1 libvulkan1 mesa-vulkan-drivers apt install libgomp1 libvulkan1 mesa-vulkan-drivers
CUDA 12 libgomp1 + NVIDIA Container Toolkit on host NVIDIA docs
ROCm libgomp1 + ROCm runtime ROCm docs

Docker Setup

CPU Variant (simplest)

Add to your container entrypoint or run once inside the container:

apt-get update -qq && apt-get install -y -qq --no-install-recommends libgomp1 && rm -rf /var/lib/apt/lists/*

Vulkan Variant (Intel / AMD GPU)

Pass the GPU device and install Vulkan libraries:

# docker-compose.yml
services:
  jellyfin:
    image: jellyfin/jellyfin
    devices:
      - /dev/dri:/dev/dri

Then inside the container:

apt-get update -qq && \
apt-get install -y -qq --no-install-recommends \
  libgomp1 libvulkan1 mesa-vulkan-drivers && \
rm -rf /var/lib/apt/lists/*

Persistent Install via Entrypoint

Libraries installed with apt inside a running container are lost on restart. To make them persistent, override the entrypoint:

# docker-compose.yml
services:
  jellyfin:
    image: jellyfin/jellyfin
    entrypoint:
      - /bin/bash
      - -c
      - |
        dpkg -s libgomp1 > /dev/null 2>&1 || \
          (apt-get update -qq && \
           apt-get install -y -qq --no-install-recommends \
             libgomp1 > /dev/null 2>&1 && \
           rm -rf /var/lib/apt/lists/*)
        exec /jellyfin/jellyfin
    devices:
      - /dev/dri:/dev/dri   # only if using GPU variants
Unraid users: You can add the apt install command to the container's Post Arguments or use the User Scripts plugin to run it on container start.

Troubleshooting

"Missing libgomp.so.1"

All whisper-cli variants use OpenMP for threading. Install it with:

apt install libgomp1

"Missing libvulkan.so.1"

The Vulkan variant needs the Vulkan loader. Jellyfin's bundled FFmpeg includes its own copy, but whisper-cli needs the system one:

apt install libvulkan1 mesa-vulkan-drivers

Binary downloads return 404

The download URL uses the plugin version to find the matching GitHub release. Make sure the plugin version matches an existing release.

Verify GPU inside Docker

# Check Vulkan
docker exec jellyfin apt-get update -qq && \
  docker exec jellyfin apt-get install -y -qq vulkan-tools && \
  docker exec jellyfin vulkaninfo --summary

# Check NVIDIA
docker exec jellyfin nvidia-smi
